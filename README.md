# CRK Party Website

## Compatriots For Reforms Kenya (CRK) Party

**Slogan:** The Urgency of Now 🚩

### About
The official website for the Compatriots For Reforms Kenya (CRK) Party - A political party committed to transforming Kenya through progressive policies, transparent governance, and citizen-centered development.

### Website Features

✅ **Home Page** - Welcoming hero section with party mission
✅ **About Section** - Party values and commitment to reform
✅ **Manifesto** - Key policy areas (Education, Economy, Healthcare, Environment, Justice, Infrastructure)
✅ **Leadership Team** - Party leadership information
✅ **News & Updates** - Latest party news and announcements
✅ **Contact Section** - Get in touch with the party
✅ **Responsive Design** - Works perfectly on mobile and desktop

### How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through different sections using the menu
4. Fill out the contact form to get in touch

### Getting Started

Simply open the `index.html` file in any modern web browser. No installation or dependencies required!
